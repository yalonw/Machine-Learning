　　張雄細說五銀


　　
　　本報記者薛原李增輝


　　“既可惜，也不可惜。”中國游泳隊今天拿下五枚
銀牌後，總教練張雄這樣評價道：“說可惜，因為在上
午的預賽中，3 個項目中國隊都有兩人闖入決賽。當時
大家想，總有一個項目可以拿到金牌吧。誰想到3 個項
目都沒有保住。說不可惜，因為今天的五個項目中國隊
原本就處於劣勢。賽前就預料到這也許是我們最‘黑’
的一天。現在看，情況還比預料的要好一點點。”


　　今天的五個項目，中國隊除了男子4 0 0 米混合泳
由老將熊國鳴擔綱外，其餘的全是新人出陣。第一項男
子1 0 0 米蝶泳，日本的山本貴司今年的最好成績是5 
3 秒3 9 ，今天又游出了5 3 秒3 4 的成績，創下了新
的亞洲紀錄。從實力講，中國隊的張宵和張強同他確實
有差距。第二項女子2 0 0 米自由泳，昨天覃彩妮在2 
0 0 米接力中游出了2 分0 秒7 3 的個人最好成績，如
果今天能保持這股勢頭，金牌就是她的了。可惜覃彩妮
是第一次參加國際大賽，同中國臺北隊的名將蔡淑敏比
起來經驗差很多。而且，她的心理壓力也比較大，最後
導致失利。


　　第三個項目男子4 0 0 米混合泳，中國隊的老將熊
國鳴遊得相當出色。但是日本隊的森隆弘年輕氣盛，最
後在體力上佔據優勢而獲勝。第四項更可惜，要論速度
，中國隊的小將齊暉其實比日本的田中雅美要快，輸就
輸在最後一個轉身上，說到底還是技術不過關。最後一
項男子4 ×2 0 0 米自由泳接力，中國隊的第二棒杜傑
遊的稍差，失利在所難免。


　　“不過。”張雄話鋒一轉，“從這兩天的比賽看，
日本隊並非賽前預料的那麼可怕。在一些我們本以為無
望的項目上，實際上中日兩隊展開了激烈的爭奪。昨天
晚上團部開會，我們確定了新的作戰思想，那就是甩開
今年已有的成績依據，一項項地和日本隊去拼。我們要
拋掉自己是亞洲游泳強國的老觀念，將自己擺在拼的位
置上去。”


　　“從這屆亞運會看，中日兩國都將目標放在了2 0 
0 0 年的悉尼奧運會上。亞運會我們派出這麼多新人出
戰，難免有閃失。中日兩國在游泳項目上的競爭會持續
很長一段時間。”


　　（本報曼谷1 2 月8 日電）
